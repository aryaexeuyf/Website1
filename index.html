Baterai + Charger
  • Baterai (+) → JST-PH2 (+) → TP4056 Bat+
  • Baterai (-) → JST-PH2 (-) → TP4056 Bat-
  • TP4056 OUT+ → Saklar → ESP32 VIN
  • TP4056 OUT- → ESP32 GND

Langkah Flashing via Android

  1. Install aplikasi ESP32 Flasher dari file ZIP.
  2. Buka aplikasi → atur file dengan offset:
    • emensta_official_jammer_esp32.bin → 0x10000
    • bootloader_kusus.bin → 0x1000
    • partition_extra.bin → 0x8000
  3. Aktifkan Auto Bootloader di aplikasi.
  4. Sambungkan ESP32 via OTG → tekan FLASH.
Pastikan HP Android ≥ 9 dan support OTG!

Troubleshooting

  • OLED tidak nyala? → Cek pin SDA/SCL (GPIO 4 & 5), pastikan firmware mendukung I2C.
  • Gagal flash? → Cek kabel data, offset, dan tipe ESP32.
  • nRF24 error? → Pastikan kapasitor 10µF terpasang dekat VCC/GND.
