Panduan Flashing Emensta BlueJammer (ESP32)

Versi: Android-friendly ‚Äî tampilan optimal untuk layar HP. Watermark: by aptech/aryaputra

1. Bahan yang diperlukan Android 9+

  • ESP32 WROOM devkit (V1‚ÄìV4) ‚Äî jangan pakai ESP32-C3/S3/non-devkit.
  • Kabel OTG data (microUSB/USB-C yang benar-benar support data).
  • 2√ó nRF24L01 (module 2.4GHz, sertakan kapasitor 10¬µF dekat VCC/GND tiap modul).
  • Li-Ion 3.7V 1000mAh, modul charger TP4056, JST connector, dan mini slide switch.
  • Optional: OLED I2C 0.96" (cek firmware harus mendukung OLED).

2. Ringkasan cepat

  1. Install app flasher Android (XAPK atau Play Store).
  2. Masukkan file di urutan: main.bin (0x10000) ‚Üí bootloader (0x1000) ‚Üí partition (0x8000).
  3. Hubungkan via OTG, lalu Flash.
  4. Jika sukses, cek Serial Monitor & OLED.

3. File di ZIP & fungsi partition

// File yang tersedia di ZIP

emensta_official_jammer_esp32.bin bootloader_kusus.bin partition_extra.bin

Apa itu partition_extra.bin? Partition table mengatur bagaimana flash dibagi: ukuran partisi untuk firmware, filesystem (spiffs/fat), NVS, OTA, dsb. Jika partition table tidak cocok, board bisa gagal booting atau data tidak tersimpan. Gunakan partition_extra.bin yang sudah disesuaikan untuk devkit WROOM (tested).

4. Cara atur file di aplikasi flasher (Android)

Slot 1: emensta_official_jammer_esp32.bin  ‚Äî offset 0x10000

Slot 2: bootloader_kusus.bin                  ‚Äî offset 0x1000 Slot 3: partition_extra.bin                   ‚Äî offset 0x8000 Centang: Auto bootloader (jika tersedia)

Perhatikan urutan & offset. Salah offset = gagal booting.

5. Langkah flashing & pengecekan

  1. Hubungkan HP & ESP32 via OTG (pastikan HP deteksi device, beri permission bila muncul).
  2. Tekan Flash dan tunggu progres selesai (jangan cabut kabel).
  3. Buka Serial Monitor untuk melihat pesan boot. Jika muncul teks boot, firmware berjalan.
  4. Jika OLED tidak menyala namun serial boot muncul => wiring OLED salah.

6. Troubleshooting singkat

  • Flashing error: Ganti kabel OTG, cek permissions, coba opsi baud lebih rendah.
  • Device tidak boot: Cek offset & partition table.
  • NRF24 problem: Pastikan kapasitor 10¬µF di VCC-GND tiap module & jalur 3.3V stabil.
  • OLED mati: Periksa SDA/SCL (lihat tabel wiring di bawah).
